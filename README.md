# Inspirize Writing Prompt Generator Version 0.0
A web app to generate random, customizable prompts for writers who are looking for a warmup or inspiration.  
Created by Levi Zelley on 12/27/2025.

## Site Use
The site is intended to be found at https://inspirize.acva.dev/ and is a simple web app accessible by anyone.  
Functionality includes checkboxes to adjust site and prompt settings, a button to generate a prompt, and local storage to save dark mode preferences.
